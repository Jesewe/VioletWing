# Code of Conduct

## Our standards

VioletWing is a technical project. Keep interactions focused, respectful, and on-topic.

**Expected behavior:**

- Treat everyone with basic respect, regardless of experience level.
- Give and receive feedback on the work, not the person.
- Accept maintainer decisions on scope and direction, even when you disagree.
- Keep discussion in public channels unless the topic is sensitive (security, harassment).

**Not acceptable:**

- Harassment, personal attacks, or sustained negative behavior toward any participant.
- Sexualized language or imagery in any project space.
- Sharing someone's private information without their consent.
- Spam, off-topic promotion, or attempts to derail technical discussion.

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any communication that represents the project publicly.

## Enforcement

Report violations to [jesewescience@outlook.com](mailto:jesewescience@outlook.com). Reports are reviewed confidentially. Include screenshots or links if you have them.

Maintainers may remove comments, close issues, or restrict access without prior warning for clear violations. Repeat or severe violations result in a permanent ban.

### Enforcement levels

| Level         | Trigger                                  | Consequence                                   |
| ------------- | ---------------------------------------- | --------------------------------------------- |
| Warning       | First offense, minor                     | Private message explaining the problem        |
| Temporary ban | Repeated minor or single serious offense | 30–90 day restriction from all project spaces |
| Permanent ban | Severe or persistent violations          | Permanent removal from all project spaces     |
